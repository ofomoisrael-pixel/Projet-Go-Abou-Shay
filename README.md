⚔️ Projet RED - CLI RPG en Go

Bienvenue dans Projet RED, un jeu de rôle textuel développé en Golang. Ce projet a été réalisé dans le cadre de l'Ytrack chez Ynov.
📝 Présentation

Le jeu vous permet de créer un héros personnalisé parmi plusieurs classes (Humain, Elfe, Nain, et même des classes spéciales comme Rayann ou Hollow). Vous devrez gérer votre inventaire, apprendre des sorts puissants et affronter des monstres dans des combats stratégiques au tour par tour.
Fonctionnalités principales :

    Création de personnage : Choix de la classe et formatage automatique du nom.

    Système d'inventaire : Gestion des consommables (potions de vie et de poison) et des livres de sorts.

    Combat au tour par tour : Affichage dynamique des tours, des points de vie (PV) restants et patterns d'attaque des monstres.

    Économie : Système de monnaie (Gold) pour acheter des objets chez le marchand.

    Gestion de la mort : Système "Wasted" avec résurrection à 50% de la vie maximum.

🚀 Installation et Lancement
Prérequis

    Avoir Go installé sur votre machine (version 1.21 ou supérieure recommandée).

Instructions

    Cloner le dépôt :
    Bash

    git clone [LIEN_DE_TON_DEPOT]
    cd [NOM_DU_DOSSIER]

    Se déplacer dans le dossier source :
    Bash

    cd src

    Lancer le jeu :
    Bash

    go run .

📂 Structure du Dépôt

    /src : Contient tous les fichiers sources .go (logique du jeu, combat, inventaire).

    /docs : Contient le document de gestion de projet et le suivi des tâches.

    README.md : Ce fichier d'instructions.
